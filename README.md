FOV changer

解凍して置く場所はどこでもOKです

起動時にPythonが必要です v3.12以上(最新でOK)
Pythonがインストールされてなければ先にインストールから初めて下さい

https://www.python.org/downloads/

終わったら dw8e_camera_editor_mod.py を起動します
管理者権限で起動しますか？など表示されるので はい でOKです

ウィンドウが開いたらゲームを起動してあとはウィンドウの下にある使い方を見てください

1. ゲームを起動し、「接続」ボタンでプロセス接続をする
2. プロセス接続の状態が「接続中」になっていたら、FOV Preset からFOVを選択するかvalの値を自分で設定するかはおまかせします
3. 青色の「全て適応(バトルスタートしたら押す)」ボタンを押す。FOVが即座に変わります

※ Offsetは使い方が特に分からない限り値を変更しないでください
メモリスキャナーは開発のために個人で実装した機能で、通常使うことはありません。

EN
This mod solves the problem of the field of view being too narrow in very old games where the FOV cannot be changed.
It also solves the issue where the camera faces forward when guarding, making it difficult to see.

Download https://github.com/overmase/-FOV-Changer/releases/tag/mod

FOV changer

You can place the unzipped file anywhere.

Python is required to start the program. v3.12 or higher (latest version is OK).

If Python is not installed, please start by installing it first.

https://www.python.org/downloads/

Once finished, run dw8e_camera_editor_mod.py.
It will ask if you want to run it with administrator privileges, so select Yes.

Once the window opens, start the game and then refer to the instructions at the bottom of the window.

1. Start the game and connect the process using the "Connect" button.
2. If the process connection status is "Connecting," you can either select an FOV from FOV Preset or set the value of val yourself.
3. Press the blue "Apply All (Press after battle starts)" button. The FOV changes instantly.

* Do not change the Offset value unless you are particularly familiar with its use.

The memory scanner is a feature implemented personally for development purposes and is not normally used.

FOV 調整器

您可以將解壓縮後的檔案放置在任意位置。

運行此程式需要 Python，版本 3.12 或更高（最新版本即可）。

如果尚未安裝 Python，請先安裝。

https://www.python.org/downloads/

安裝完成後，執行 dw8e_camera_editor_mod.py。

程式會詢問您是否要以管理員權限執行，請選擇「是」。

視窗打開後，啟動遊戲，然後參考視窗底部的說明。

1. 啟動遊戲，然後點選「連線」按鈕連線進程。

2. 如果進程連線狀態為“正在連線”，您可以從 FOV 預設中選擇一個 FOV，或自行設定 val 值。

3. 按下藍色的「全部套用（戰鬥開始後按下）」按鈕。 FOV 將立即變更。

* 除非您非常熟悉其用法，否則請勿變更偏移值。

記憶體掃描器是為開發目的而個人實現的功能，通常不會使用。
