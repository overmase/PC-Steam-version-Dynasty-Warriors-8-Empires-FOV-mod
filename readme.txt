Installation Instructions

FOV Changer

You can place the unzipped file anywhere.

Python is required to start the program. Version 3.12 or later (latest version is fine).

If Python is not installed, please install it first.

https://www.python.org/downloads/

Once installed, run dw8e_camera_editor_mod.py.

You will be asked if you want to run it with administrator privileges; select Yes.

Once the window opens, start the game and then refer to the instructions at the bottom of the window.


安裝說明

FOV 切換器

您可以將解壓縮後的檔案放置在任意位置。

執行此程式需要 Python，版本 3.12 或更高版本（最新版本即可）。

如果尚未安裝 Python，請先安裝。

https://www.python.org/downloads/

安裝完成後，執行 dw8e_camera_editor_mod.py。

系統會詢問您是否要以管理員權限執行，請選擇「是」。

視窗打開後，啟動遊戲，然後參考視窗底部的說明。
